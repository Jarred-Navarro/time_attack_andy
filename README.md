# time_attack_andy
A speedrunning 2D platformer, written using the Arcade library in Python.

Background Music: "Jungle Driver" by Waterflame...https://www.newgrounds.com/audio/listen/1123551

Coin Collect Sound: Arcade Built in Resources...
Death Sound: Arcade Built in Resources...https://api.arcade.academy/en/development/api_docs/resources.html#

Font: Public Pixel by GGBotNet...https://ggbot.itch.io/public-pixel-font

Background Tiles: Pixel Platformer (Assorted) by KennyNL...https://kenney.nl/assets/series:Pixel%20Platformer?sort=update

Player Textures: Drawn by Me in Microsoft Paint and Paint.Net
Portal Texture: Drawn by Me in Microsoft Paint


All links accurate as of 5/11/2025

Thank you for playing my game! :) I hope you enjoy!
